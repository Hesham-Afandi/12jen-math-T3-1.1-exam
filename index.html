Rational Expressions: Simplifying, Operations & Solving Equations

📅 التاريخ: 👤 الاسم: 🏫 الشعبة: ⏱️ المدة: 60 دقيقة 📊 المجموع: 100 درجة

🔹 الجزء الأول: تبسيط التعابير النسبية والقيم الممنوعة

  1. 1. التعبير النسبي (Rational Expression) هو:
    ✅ الإجابة: أ) كسر بسطه ومقامه كثيرات حدود، مثل (x²-1)/(x+2)
  2. 2. القيم الممنوعة (Restricted Values) للتعبير (x+3)/(x²-9) هي:
    ✅ الإجابة: ج) x²-9 = (x-3)(x+3) = 0 → x = 3 أو x = -3
  3. 3. عند تبسيط التعبير (x²-4)/(x+2)، الناتج هو:
    ✅ الإجابة: ب) (x²-4)/(x+2) = (x-2)(x+2)/(x+2) = x - 2
  4. 4. لماذا لا يمكن أن يساوي المقام صفراً في التعبير النسبي؟
    ✅ الإجابة: ب) القسمة على صفر غير معرّفة في الرياضيات

🔹 الجزء الثاني: ضرب وقسمة التعابير النسبية

  1. 5. عند ضرب تعبيرين نسبيين، نقوم بـ:
    ✅ الإجابة: أ) نضرب البسط × البسط والمقام × المقام ثم نختصر
  2. 6. ناتج ضرب (x+1)/(x-2) × (x-2)/(x+3) هو:
    ✅ الإجابة: أ) نختصر (x-2) من البسط والمقام، فيتبقى (x+1)/(x+3)
  3. 7. عند قسمة تعبيرين نسبيين، نقوم بـ:
    ✅ الإجابة: ب) نضرب الكسر الأول في مقلوب الكسر الثاني
  4. 8. ناتج قسمة (x²-1)/(x+2) ÷ (x+1)/(x-3) هو:
    ✅ الإجابة: أ) = (x²-1)/(x+2) × (x-3)/(x+1) = (x-1)(x+1)(x-3)/[(x+2)(x+1)] = (x-1)(x-3)/(x+2)

🔹 الجزء الثالث: جمع وطرح التعابير النسبية

  1. 9. عند جمع تعبيرين نسبيين لهما نفس المقام، نقوم بـ:
    ✅ الإجابة: أ) نجمع البسوط ونكتب المقام المشترك
  2. 10. ناتج جمع 3/x + 5/x هو:
    ✅ الإجابة: أ) (3+5)/x = 8/x
  3. 11. عند جمع تعبيرين نسبيين بمقامات مختلفة، الخطوة الأولى هي:
    ✅ الإجابة: ب) يجب إيجاد LCD أولاً لتوحيد المقامات
  4. 12. المقام المشترك الأصغر (LCD) للتعبيرين 1/(x-2) و 1/(x+3) هو:
    ✅ الإجابة: ب) LCD = (x-2)(x+3) لأن المقامين لا يحتويان على عوامل مشتركة

🔹 الجزء الرابع: حل المعادلات النسبية

  1. 13. الخطوة الأولى لحل معادلة نسبية هي:
    ✅ الإجابة: أ) يجب أولاً تحديد القيم الممنوعة التي تجعل المقام = 0
  2. 14. حل المعادلة 2/x = 4/(x+2) هو:
    ✅ الإجابة: أ) بالضرب التبادلي: 2(x+2) = 4x → 2x+4 = 4x → 4 = 2x → x = 2
  3. 15. حل المعادلة 3/(x-1) = 6/(x+1) هو:
    ✅ الإجابة: أ) بالضرب التبادلي: 3(x+1) = 6(x-1) → 3x+3 = 6x-6 → 9 = 3x → x = 3
  4. 16. عند حل معادلة نسبية، ما الذي يجب فحصه في النهاية؟
    ✅ الإجابة: أ) يجب التأكد من أن الحل ليس قيمة ممنوعة

🔹 الجزء الخامس: مسائل تطبيقية

  1. 17. بسّط التعبير (x²-16)/(x²-8x+16):
    ✅ الإجابة: أ) (x²-16) = (x-4)(x+4) و (x²-8x+16) = (x-4)² → (x+4)/(x-4)
  2. 18. القيم الممنوعة للتعبير (x+5)/(x²-25) هي:
    ✅ الإجابة: ج) x²-25 = (x-5)(x+5) = 0 → x = 5 أو x = -5
  3. 19. ناتج جمع 2/(x+1) + 3/(x-1) هو:
    ✅ الإجابة: أ) LCD = (x+1)(x-1) = x²-1 → [2(x-1)+3(x+1)]/(x²-1) = (5x+1)/(x²-1)
  4. 20. حل المعادلة 5/x + 2/(x-3) = 1 هو:
    ✅ الإجابة: ب) LCD = x(x-3) → 5(x-3)+2x = x(x-3) → 7x-15 = x²-3x → x²-10x+15=0 → بالحل: x = 1 أو x = 15 (كلاهما ليس قيمة ممنوعة)
